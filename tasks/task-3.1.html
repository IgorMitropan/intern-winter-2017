Task 3.1

Условие:

Имеется следующее описание:
        var
            a, b, c: set of Boolean;
            c: Real;
            w: Integer;
            m, n: Boolean;

Какие из этоих переменных совместимы по присваиванию?

Решение:

Поскольку в JavaScript при объявлении переменных не объявляется ее тип данных,
то любые переменные совместимы по присвоению. 

Например следующий snippet не приведет к ошибке:
    let set = new Set();
    set = 11.1;
    set = 11;
    set = true;
