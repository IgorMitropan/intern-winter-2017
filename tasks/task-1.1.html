Task 1.1

Условие:

В следующем фрагменте программы выполните оптимизацию:

        for i := 1 to bound do
        begin
            for j := 1 to dim do
            begin
            s := 0.0;

                for k := 0 to i - 1 do
                begin
                s := s + dt * weight[i, k] * slope[k, j];
                z[i, j] := x0[j] + 2.314 * y;
                end;
            end;

            F(t0 + abscissa[i] * dt, z[i], slope[i]);
        end;

Решение:

Код преобразован в JavaScript:
        s = 0;                              // s приравнивается 0 на каждом шаге цикла for по j, но в теле цикла нигде не используется, вынес из цикла.

        for (let i = 1; i < bound; i++) {
            for (let j = 1; j < dim; j++) {
                z[i][j] = x0[j] + 2.314 * y; // Эту строку вынес из цикла for по k, т.к. на каждом шаге этого цикла значение перезаписывается на точно такое же
            } 
                                            // поскольку цикл for по k изменял значение переменной s то его вообще можно удалить (Читай комментарий к строке 1)
            F(t0 + abscissa[i] * dt, z[i], slope[i]);      
        }
